# querytoExcel-dotnet


#How TO USE


1. Download "QueryToExcel dotnet.zip"
2. Unzip it
3. run the .exe inside the folder
4. input the information of "Server Name", "Username", "Password", and "Database Name" with the information of your SQL Server
